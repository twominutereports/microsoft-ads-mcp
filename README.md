## Why Use This

The Microsoft Ads platform is powerful, but pulling keyword-level performance, comparing campaigns, and building cross-channel reports means navigating a clunky UI and exporting spreadsheets. This MCP server cuts through all of that.

**What you get:**

- **Instant search ad analytics** — ask about CPC, CTR, conversions, and ROAS across campaigns and keywords without pulling manual reports.
- **No setup complexity** — no Microsoft Advertising API developer credentials required; connect through Two Minute Reports in minutes.
- **Multi-account support** — analyze multiple Microsoft Ads accounts from a single conversation.
- **Cross-platform context** — combine with 22+ other sources like Google Ads, Google Analytics 4, and HubSpot to compare search channel performance side by side.
